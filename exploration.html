What is the structure of your dataset?¶

  • 有328553个标，8个特性（包括借款金额，借款期限，借款利率，初始评级，逾期率，借款成功日期，借款类型，是否首标）
  • 借款金额，借款利率，逾期率是数值类型的
  • 初始评级是分类，按以下排序(好->坏)：A，B，C，D，E，F
  • 借款期限是分类，按以下排序：(0-6]，(6-12]，(12-18]，(18-24]
  • 借款类型是分类，没有顺序
  • 是否首标是分类，没有顺序
  • 借款成功日期是datetime类型

# log变换(10为底)
def trans(x, inverse = False):
    if not inverse: return np.log10(x)
    else: return 10 ** x
# 创建log变换后的数据

bin_log_edges = np.arange(trans(100), trans(500000)+0.5, 0.5)
bin_edges = 10 ** bin_log_edges
bin_centers = 10 ** (bin_log_edges + 0.5/2)[:-1]

def my_errorbar(data,label,color):
    data = data[data['借款期限']==label]
    # pd.cut分箱
    data_bins = pd.cut(data['借款金额'], bin_edges, right = False, include_lowest = True)
    # # 计算每个分组的统计量(均值)
    y_means = data['借款利率'].groupby(data_bins).mean()
    y_std = data['借款利率'].groupby(data_bins).std()
    # # 绘制汇总数据图
    plt.errorbar(x = bin_centers, y = y_means, color=color, label=label, alpha=0.5);

label = df_lc_clean['借款期限'].value_counts().index
color = ['r','b','g','y']
for j,k in zip(label, color):
    my_errorbar(data=df_lc_clean, label=j, color=k)

plt.xscale('log')
plt.xticks(bin_edges,np.round(bin_edges/100,1))
plt.xlabel('借款金额 x1000');
